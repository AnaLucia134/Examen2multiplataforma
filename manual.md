# Manual de Usuario

## Aplicación TODO LIST

Esta aplicación permite al usuario gestionar sus tareas de forma sencilla desde su dispositivo móvil.

---

## Pantalla principal

Al ingresar a la aplicación, se muestra la lista de tareas registradas.

Desde esta pantalla el usuario puede:
- Ver todas sus tareas
- Ver cuántas tareas están pendientes
- Marcar tareas como completadas
- Eliminar tareas
- Presionar el botón **+ Nuevo** para agregar una tarea

---

## Agregar tarea

Para agregar una nueva tarea:

1. Presionar el botón **+ Nuevo**
2. Completar los campos:
   - Título de la tarea
   - Materia
   - Fecha de vencimiento (formato YYYY-MM-DD)
   - Estado inicial
3. Presionar **Guardar**

La tarea aparecerá automáticamente en la lista principal.

---

## Detalle de tarea

Al presionar una tarea, se abre su detalle.

En esta pantalla se puede:
- Editar el título
- Editar la materia
- Modificar la fecha
- Cambiar el estado de la tarea
- Guardar los cambios

Al guardar, se regresa automáticamente a la pantalla principal.

---

## Estados de las tareas

Las tareas pueden tener dos estados:
- Pendiente
- Completada

Las tareas pendientes aparecen primero en la lista.

---

## Notas importantes

- Las tareas se guardan automáticamente
- No se necesita conexión a internet
- Los datos no se pierden al cerrar la aplicación
